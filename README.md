# PROYECTO BLOG INFORMATIVO SALUD MENTAL

## Descripción

Este repositorio contiene la planificación del proyecto "Blog Informativo sobre Salud Mental", desarrollado como parte de la evidencia del programa ADSO del SENA.

## Roles del equipo:
1️⃣ Product Owner: Define los requisitos, decide qué funcionalidades son más importantes y prioriza el Backlog.
2️⃣ Scrum Master: Organiza las reuniones, elimina bloqueos y supervisa el cumplimiento del Sprint.
3️⃣ Desarrollador Frontend: HTML, CSS, JavaScript y diseño de la interfaz
4️⃣ Desarrollador Backend: Manejo de usuarios, datos, API simulada.
5️⃣ Tester: Pruebas del sistema y reporte de errores.

## Definición del proyecto y necesidades

El proyecto consiste en desarrollar un Blog Web de Salud Mental, cuyo objetivo es brindar información clara y confiable sobre temas relacionados con el bienestar emocional, la ansiedad, el estrés, la autoestima, la inteligencia emocional y otros temas de interés para la comunidad.
El blog permitirá la publicación de artículos organizados por categorías, la visualización de contenido de manera sencilla y una interfaz amigable para el usuario.

## Necesidades identificadas
Facilitar el acceso a información sobre salud mental.
Organizar el contenido por categorías.
Permitir una navegación sencilla.
Crear una interfaz atractiva y accesible.
Promover hábitos de autocuidado.
Ofrecer una plataforma adaptable a dispositivos móviles.


1️⃣ Planeación:
